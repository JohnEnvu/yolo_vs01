   用在变量，函数
volatile - 禁止编译器优化，使用就从内存读（正常是从寄存器读，临时）
   硬盘 - 内存 - 寄存器
mutable - 只读中可写
```cpp
class EBook {
    int readCount; // 普通变量
public:
    // const 函数：承诺不修改书的价格等核心信息
    void read() const {
        // 报错！保安拦住你：你说过 read() 是只读的，不许改 readCount！
        readCount++; 
    }
};
// 对比
class EBook {
    mutable int readCount; // 特许：即使在只读环境下也可以修改
public:
    void read() const {
        // 成功！保安看到标记，允许修改 readCount
        readCount++; 
        std::cout << "正在阅读，已读 " << readCount << " 次";
    }
};
```
