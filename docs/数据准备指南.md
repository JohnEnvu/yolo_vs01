# 数据集准备指南

## 1. 数据收集 (Data Collection)
你需要收集包含以下目标的图片（建议至少 500-1000 张）：
*   **安全帽 (helmet)**
*   **反光衣 (vest)**
*   **工人 (person)** (可选，看具体需求)

**来源建议**：
*   网络爬虫 (百度/Google 图片)
*   公开数据集 (如 Hard Hat Workers Dataset)
*   自己拍摄 (如果有施工现场条件)

将所有图片放在 `data/images` 目录下（可以先建一个临时文件夹存放）。

## 2. 数据标注 (Data Annotation)
我们使用 **LabelImg** 工具进行标注。

### 安装 LabelImg
在你的 conda 环境中：
```bash
pip install labelimg
labelimg
```

### 标注步骤
1.  打开 LabelImg。
2.  点击 "Open Dir" 选择你的图片文件夹。
3.  点击 "Change Save Dir" 选择标签保存文件夹（通常与图片同级，或新建 `labels` 文件夹）。
4.  **重要**：在左侧工具栏确保格式选为 **YOLO** (不是 PascalVOC)。
5.  开始标注：
    *   按 `w` 键画框。
    *   输入类别名称 (如 `helmet`, `vest`)。
    *   按 `d` 键切换下一张。

## 3. 数据集划分 (Data Split)
标注完成后，你需要按比例（如 8:1:1）将数据划分为训练集、验证集和测试集。

最终目录结构应如下：
```
data/
├── images/
│   ├── train/  (存放80%图片)
│   ├── val/    (存放10%图片)
│   └── test/   (存放10%图片)
└── labels/
    ├── train/  (对应的txt标签)
    ├── val/    (对应的txt标签)
    └── test/   (对应的txt标签)
```

## 4. 创建数据集配置文件
在 `data` 目录下创建一个 `construction.yaml` 文件：

```yaml
path: ../data  # 数据集根目录
train: images/train
val: images/val
test: images/test

names:
  0: helmet
  1: vest
  2: person
```
