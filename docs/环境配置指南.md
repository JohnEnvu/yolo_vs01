# 环境配置与项目搭建指南

## 1. 准备工作
由于你的电脑上 `conda` 命令没有直接添加到系统路径，你需要使用 **Anaconda Prompt** 来执行命令。

1.  点击 Windows 开始菜单。
2.  搜索并打开 **Anaconda Prompt** (或者 Miniconda Prompt)。
3.  在打开的黑色窗口中，通过输入盘符切换到项目所在硬盘（假设项目在 G 盘）：
    ```bash
    G:
    ```
4.  进入项目目录：
    ```bash
    cd g:\work\yolo0213\v8
    ```

## 2. 创建并激活 Conda 环境
我们要创建一个名为 `yolo_env` 的干净环境，Python 版本选择 3.9（兼容性最好）。

```bash
# 1. 创建环境 (中间需要输入 y 确认)
conda create -n yolo_env python=3.9 -y

# 2. 激活环境
conda activate yolo_env
```

## 3. 安装 YOLOv8 源码 (核心步骤)
我们要以“开发者模式”安装刚才下载的 YOLOv8 源码，这样你以后修改源码（比如改模型结构）才会生效。

```bash
# 1. 进入源码目录
cd train_project/ultralytics_repo

# 2. 执行安装 (注意最后有个点 ".")
# 这会自动安装 torch, torchvision, matplotlib, opencv 等所有深度学习依赖
pip install -e .
```

## 4. 安装界面开发依赖
YOLO 源码安装好后，我们还需要安装 PyQt5 来开发上位机界面。

```bash
# 1. 回到项目根目录
cd ../..

# 2. 安装 PyQt5
pip install PyQt5
```

## 5. 验证环境
一切安装完成后，运行以下命令检查是否成功：

```bash
# 检查 YOLO 是否来自本地源码
python -c "import ultralytics; print('YOLO路径:', ultralytics.__file__)"

# 检查 PyQt5 是否可用
python -c "import PyQt5; print('PyQt5已安装')"
```

如果输出的 YOLO 路径包含 `train_project`，说明配置完美！
